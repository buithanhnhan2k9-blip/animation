Bài 3: GSAP Basic Tween

GSAP Move

Sử dụng gsap.to() để thay đổi vị trí và độ mờ của phần tử khi trang được tải. 

gsap.to(".gsap-box", {
  x: 200,
  opacity: 1,
  duration: 1.5
});
MENU HEADER

Nội Dung Chính

Bài này sử dụng GSAP Timeline để tạo chuỗi animation chạy lần lượt theo thứ tự. 

const tl = gsap.timeline();
tl.from(".header-anim",{y:-50})
  .from(".hero-title",{opacity:0})
  .from(".hero-cta",{scale:0});

Khi cuộn tới section này, GSAP sử dụng stagger để các sản phẩm xuất hiện lần lượt. 

gsap.to(".card", {
  opacity: 1,
  y: 0,
  stagger: 0.2
});
